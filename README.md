# Digital Payment System (C)

## Description
A console-based digital payment system built in C. It allows users to create accounts, check balance, transfer money, and view transaction history.

## Features
- Account creation  
- Balance checking  
- Money transfer  
- Transaction history  

## Technologies Used
- C Programming  
- File Handling  

## How to Run
1. Compile the code using any C compiler  
2. Run the program  
3. Follow the menu options
