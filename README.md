# Digital Payment System (C)

## Description
A console-based digital payment system inspired by mobile financial services like Nagad. The system allows users to perform common financial operations such as sending money, cash out, mobile recharge, bill payment, and balance checking. User data and transaction history are managed using file handling.

## Features
- Cash Out  
- Send Money  
- Mobile Recharge  
- Bill Payment  
- EMI Payment  
- Balance Enquiry  
- Mini Statement (Transaction History)  
- PIN Reset & Change  

## Technologies Used
- C Programming  
- File Handling  
- Structured Programming  

## How It Works
- User enters a dial code (*167#) to access the system  
- PIN authentication is required for transactions  
- Balance and PIN are stored in a file (`PinBalance.txt`)  
- Transaction history is stored in (`trids.txt`)  

## How to Run
1. Open the project in any C compiler (CodeBlocks / VS Code)  
2. Compile the program  
3. Run the executable file  
4. Enter `*167#` to start and follow menu options  

## Notes
- This is an academic project created to simulate basic digital payment system functionality  
- File handling is used instead of a database
